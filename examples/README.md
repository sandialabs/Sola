# Examples

The examples are organized as follows:

 - `from_publications/`: These are examples that are in publications. Each example has a "Article_Reference.txt" file that identifies the publication that is appears in.

 - `in_development/`: These are examples being actively developed. They will eventually be moved to "in_publications" if they appear in a publication or "inactive" otherwise.

 - `inactive/`: These are examples that were developed to reasonable maturity, but where never used in a publication and are no longer being actively developed.

 - `experimental/`: These are examples that were put together for a quick experiment, but not developed to full maturity. They may move to "in_development" if the quick experiment is fruitful and leads for more detailed analysis.
