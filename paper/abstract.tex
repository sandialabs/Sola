This paper presents a software framework for implementing and deploying hyper-differential sensitivity analysis, a recent approach to post-optimality for large-scale PDE-constrained optimization problems.
The framework consists of two complementary elements: \SABL, a \MATLAB library for rapid prototyping and solving small-to-modest problems, and \MrHyDE, a \cpp library equipped with accessible high-performance computing capabilities.
An elementary example demonstrates how \SABL as a high-level, mathematically intuitive library prepares users for the lower-level but more computationally powerful \MrHyDE library, and the paper concludes with a computationally demanding example facilitated by \MrHyDE.
The presented high-to-low-level paradigm can be applied to many outer loop analyses.
